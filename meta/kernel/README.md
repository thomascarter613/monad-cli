# Monorepo Kernel

The monorepo kernel is the core governance logic that defines workspace identity, invariants, mutation safety, and policy enforcement.

In Monad, the kernel is implemented through:

- `monad.toml`
- `monad.lock`
- `.monorepo.json`
- `crates/monad-core`
- `crates/monad-plans`
- `governance/*`
