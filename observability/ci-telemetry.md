# CI Telemetry

CI telemetry records the health and performance of the repository automation system.

## Signals

- Workflow duration
- Job duration
- Cache hit/miss rates
- Dependency hygiene findings
- Drift findings
- Policy findings
- SBOM generation result
- Static analysis result

## Artifacts

CI workflows should upload useful artifacts when available:

- graph reports
- drift reports
- SBOM files
- policy findings
- dependency hygiene reports
