# Artifacts (Governance‑Grade)

Artifacts in Monad are governed institutional outputs produced by deterministic execution of the lifecycle graph, workspace model, and plan‑backed mutation engine.

Artifacts are:

- **Deterministic** — reproducible from workspace + manifest + plan.
- **Governed** — subject to policy, invariants, and trust boundaries.
- **Traceable** — linked to ADRs, work packets, releases, and provenance.
- **Replayable** — reconstructable via lifecycle graph replay.
- **Auditable** — validated through artifact evidence and governance checks.

## Artifact Categories

- Build artifacts  
- Plan artifacts  
- Graph artifacts  
- Release artifacts  
- Evidence artifacts  
- Machine‑readable schemas  
- Workspace‑derived state artifacts  

## Artifact Governance Layers

- Lifecycle governance  
- Provenance ledger  
- Artifact safety envelope  
- Release evidence corpus  
- Policy‑encoded constraints  
- Retention and archival rules  

This directory contains the institutional documentation for all artifact types.
