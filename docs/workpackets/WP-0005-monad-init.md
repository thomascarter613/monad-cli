# WP-0005: monad init

## 1. Purpose

Implement the governed workspace initializer using the plan engine and approved Monad defaults.

## 2. Outcome Summary

After this work packet is complete:

- monad init creates a governed workspace.
- Dry-run writes nothing.
- Generated workspace passes check.
