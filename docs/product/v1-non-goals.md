# Monad v1 Non-Goals

The following are not required for v1:

- Dynamic Rust plugin loading.
- Remote plugin registry.
- Remote template registry.
- Hosted service.
- GUI.
- Daemon mode.
- AI code generation.
- Cloud sync.
- Complex query language.
- Perfect support for every language.
- Automatic publishing to every ecosystem.
- Deep AST import rewriting for every language.
- Full package manager abstraction across all ecosystems.
