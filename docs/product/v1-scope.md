# Monad v1 Scope

Monad v1 is the point where Monad's core contract becomes stable.

Monad v1 can:

- Initialize a governed monorepo.
- Add and manage workspace units.
- Inspect, check, graph, document, and explain the repo.
- Plan, diff, and apply safe changes.
- Generate useful context for humans and AI agents.
- Enforce baseline governance policies.
- Manage ADR and work packet artifacts.
- Dogfood its own repository.
- Avoid silently damaging user work.

Monad v1 is not the point where Monad does everything forever.
